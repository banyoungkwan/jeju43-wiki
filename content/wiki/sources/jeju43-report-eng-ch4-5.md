# Jeju 4·3 Incident Investigation Report (English) — Chapter IV·V & Appendices

> **[번역자료]** *Jeju 4·3 Incident Truth Investigation Report — Chapter IV: Damage, Chapter V: Conclusions, and Appendices.* English Translation of 제주4·3사건 진상조사보고서(2003).
> 원본 파일: `raw/ChapterIV_V_Appendices_Jeju43Report_ENG.md` (5,777행)

[[진상조사보고서]](2003) 영문 번역본의 후반부. 피해 통계(제4장), 결론(제5장), 부록(대통령 발표문·사건 일지·통계표)을 포함한다.

## 제4장: 피해 상황 (Chapter IV: Damage)

### 인명 피해

| 지표 | 수치 |
|------|------|
| 위원회 신고 희생자 | 14,028명 |
| 추정 총 희생자 | 25,000 ~ 30,000명 |
| 토벌대에 의한 희생 | 78.1% (10,955명) |
| 무장대에 의한 희생 | 12.6% (1,764명) |
| 공란(미확인) | 9.0% (1,266명) |
| 10세 이하 어린이 | 5.8% (814명) |
| 61세 이상 노인 | 6.1% (860명) |
| 여성 | 21.3% (2,985명) |
| 예비검속·형무속 추가 희생 | 약 3,000명 |
| 군 전사자 | 약 180명 |
| 경찰 전사자 | 140명 |
| 소각 가옥 | 39,285동 |

이 통계는 "토벌대에 의해 80% 이상이 사망했다"는 미군 보고서와 일치한다. 10세 이하 어린이와 61세 이상 노인이 전체 희생의 11.9%를 차지한다는 점에서 남녀노소를 가리지 않은 과도한 진압이었음이 확인된다.

### 물적 피해

- 제주도 중산간마을 95% 이상 소각.
- 이재민 2만 명 이상이 한라산으로 피신.

### 서청 관련 통계

- 4·3 발발 이전 제주 파견: 500~700명
- 1948년 말 파견: 1,000명
- 국가유공자로 등록된 우익단체원: 639명

## 제5장: 결론 (Chapter V: Conclusions)

### 핵심 결론 (한국어 원문 기준)

1. **발발 원인**: 복합적. 3·1절 발포사건 → 서청·응원경찰 탄압 → 고문치사 → 남로당 제주도당의 수세적 무장투쟁 결정. 남로당 중앙당 지령 없음.
2. **무장대 과오**: 군경뿐 아니라 선거 관리 요원·경찰 가족 등 민간인 살해는 명백한 과오. 해주대회 참가로 유혈사태 가속화.
3. **강경진압의 책임**: 1차 책임 — 9연대장·2연대장. 최종 책임 — 이승만 대통령. 미군정·주한미군사고문단도 책임 면할 수 없음.
4. **국제법 위반**: 1948년 제노사이드협약, 1949년 제네바협정이 요구하는 민간인 보호 원칙이 제주에서 명백히 무시됨.
5. **냉전의 최대 희생지**: 진상규명을 50년 억제한 요인이기도 함.

### 국가 사과

2003년 10월 31일 노무현 대통령 공식 사과. "4·3사건으로 무고한 희생이 발생된 데 대한 정부의 사과와 희생자 명예회복, 추모사업의 적극적인 추진"을 약속. 대통령은 제주를 "인권의 상징이자 평화의 섬"으로 규정.
